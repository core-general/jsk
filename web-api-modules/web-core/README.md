# Web Core

The foundational web framework module — defines all API annotations, metadata models, auth interfaces, rendering types, and info-gathering infrastructure. This is the shared contract between server and client.

## What It Solves

- 15 annotations: `@WebPath`, `@WebGET`, `@WebPOST`, `@WebMethod`, `@WebAuth`, `@WebAuthNO`, `@WebAuthBasic`, `@WebIdempotence`, `@WebRender`, `@WebUserToken`, `@WebRedirect`, `@WebFile`, `@WebAllParams`, `@WebParamsToObject`, etc.
- Auth interfaces (`WebAuthServer`/`WebAuthClient`/`WebSecretProvider`) with static-secret implementations
- Info gathering: `WebClassInfo`, `WebMethodInfo`, `WebClassInfoProvider` — loads pre-compiled JSON metadata generated by Maven plugin
- Rendering types: JSON, JSON_PRETTY, BASE64_BYTES, RAW_STRING, RAW_BYTE_ZIPPED

## Key Details

- Only depends on `jx-services-core` (no Spring, no Servlet API) — maximally portable
- Annotation inheritance: class-level defaults, method-level overrides, "NO" annotations disable features per-method
