# JX System Info

System metrics collection — CPU load, RAM usage, and JVM memory statistics using the OSHI library.

## What It Solves

- Provides real-time CPU load, total/available RAM, and JVM heap/non-heap memory metrics
- Exposes metrics via JSK service interfaces for integration with monitoring systems

## Key Details

- Depends on the OSHI library for cross-platform hardware access
- Useful for health-check endpoints and monitoring dashboards
